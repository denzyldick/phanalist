### E0015 Lack of Cohesion of Methods (LCOM4)

LCOM4 measures the number of connected components in a class. A connected component is a set of methods and properties that are related to each other.

A class is considered cohesive if all its methods and properties are related (LCOM4 = 1). If LCOM4 > 1, the class has independent "islands" of logic and should probably be split into multiple smaller classes.

#### Non-cohesive Example (LCOM4 = 2)

```php
class UserManager {
    private $db;
    private $logger;

    // Island 1: User Management
    public function saveUser($user) {
        $this->db->execute("INSERT...");
    }

    // Island 2: Logging
    public function logError($message) {
        $this->logger->write($message);
    }
}
```

In this example, `saveUser` and `logError` don't share any properties and don't call each other. They form two separate components.

#### Cohesive Example (LCOM4 = 1)

```php
class Calculator {
    private $result = 0;

    public function add($n) {
        $this->result += $n;
    }

    public function getResult() {
        return $this->result;
    }
}
```

Here, both methods use the same property `$result`, so they are connected.
